# "A metodologia adotada seguiu os preceitos do desenvolvimento orientado ao usuário. Inicialmente, realizou-se uma pesquisa de campo com 30 participantes, cujos dados (consolidados na Tabela X) serviram como requisitos para a arquitetura do software. O sistema foi estruturado em 6 módulos educativos com um banco de 60 questões, utilizando Inteligência Artificial Generativa como ferramenta de auxílio na prototipagem rápida do código em React Native. O foco do desenvolvimento foi o MVP (Produto Mínimo Viável), priorizando a lógica de persistência de dados (AsyncStorage) e a experiência de aprendizagem (gamificação), em detrimento da complexidade de infraestrutura de backend."
